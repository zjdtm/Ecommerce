# 1. Multi-Module 프로젝트 설정
  먼저 Multi-Module이란 서로 다른 즉, 독립된 프로젝트를 하나로 묶은 것을 뜻하며 이렇게 서로 독립된 프로젝트는 기능별로 명확하게 분리되어 있을 뿐더러,
  공통된 코드를 여러 곳에서 참조가 가능하기 때문에 기능 수정시 모든 프로젝트를 수정할 필요가 없다는 장점 등을 가지고 있다.
